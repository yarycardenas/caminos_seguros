# Caminos Seguros

## Descripción

Este es un proyecto Caminos seguros, tiene como objetivo principal crear una aplicación que permita a los usuarios de Villeta y alrededores a encontrar rutas seguras, reportar incidentes de seguridad y visualizar mapas de calor basados en reportes comunitarios.

La aplicación busca promover la seguridad vial y peatonal mediante el uso de tecnología móvil, facilitando la toma de decisiones informadas para evitar zonas de riesgo.

## Características Principales

- **Registro e Inicio de Sesión**: Creación de cuentas de usuario para acceder a la aplicación.
- **Reportar Incidentes**: Permite a los usuarios reportar ubicaciones inseguras o incidentes en tiempo real.
- **Visualización de Rutas**: Sugerencias de rutas seguras basadas en datos recopilados.
- **Mapa de Calor**: Visualización de áreas con mayor concentración de reportes de incidentes.
- **Perfil de Usuario**: Gestión personal de la cuenta y reportes realizados.

## Estructura del Proyecto

El proyecto está organizado en las siguientes fases:

- **1-project/**: Contiene la documentación del proyecto, incluyendo cronograma de actividades, diseño del prototipo, recursos necesarios y resultados esperados.
- **2-desing/**: Incluye diseños gráficos como logos y elementos visuales para la aplicación.
- **3-development/**: Contiene las pantallas del prototipo de la aplicación, representadas en imágenes de interfaz de usuario.

## Tecnologías Utilizadas

- **Diseño**: Herramientas de diseño gráfico Figma para crear los logos y pantallas.
- **Desarrollo**: (Pendiente) Se planea utilizar tecnologías como frameworks web como React.js, con integración de mapas Google Maps API.
